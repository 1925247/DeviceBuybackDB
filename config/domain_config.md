# Centralized Domain Configuration

This configuration file is responsible for managing all domain settings and environment variables for:
- example.com
- console.example.com
- partner.example.com

## Configuration

- BASE_URL: `https://example.com`
- CONSOLE_URL: `https://console.example.com`
- PARTNER_URL: `https://partner.example.com`

## Environment Variables

- API_KEY: your_api_key_here
- DATABASE_URL: your_database_url_here
- SECRET_KEY: your_secret_key_here

## Usage

Make sure to replace `your_api_key_here`, `your_database_url_here`, and `your_secret_key_here` with the actual values in your environment.
